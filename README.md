# glasfein-nienburg

Relaunch der Glasfein-Website als statische Mehrseiten-Site mit Startseite, Kontakt, Karriere sowie Impressum und Datenschutz.

## Struktur

- `index.html` Startseite
- `kontakt.html` Kontaktseite
- `karriere.html` Karriereseite
- `impressum.html` Impressum
- `datenschutz.html` Datenschutz
- `style.css` zentrales Styling
- `script.js` Navigation, Scroll-Reveal und Formularlogik
- `assets/` Bilder und lokal eingebundene Schriftdateien

## Entwicklung

Die Seite ist als statisches Projekt aufgebaut und kann über jeden einfachen HTTP-Server lokal ausgeliefert werden.
